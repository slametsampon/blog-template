Template proyek untuk mahasiswa dengan fokus pada pengembangan web menggunakan Next.js:

1. **Judul Proyek:**

   - Sistem Informasi Akademik Online

2. **Deskripsi Singkat:**

   - Proyek ini bertujuan untuk membuat platform Sistem Informasi Akademik yang dapat diakses secara online oleh mahasiswa, dosen, dan admin.

3. **Fitur Utama:**

   - Registrasi dan login mahasiswa, dosen, dan admin.
   - Tampilan jadwal kuliah dan pengumuman.
   - Pengelolaan data mahasiswa dan dosen.
   - Penilaian dan input nilai secara online.

4. **Teknologi yang Digunakan:**

   - Next.js untuk pengembangan front-end.
   - Serverless functions untuk backend (menggunakan API Routes di Next.js).
   - Basis data MongoDB untuk menyimpan data mahasiswa dan dosen.
   - Autentikasi menggunakan JSON Web Tokens (JWT).

5. **Struktur Proyek:**

   - `/pages`: Halaman Next.js seperti `index.js`, `login.js`, `dashboard.js`.
   - `/components`: Komponen React yang dapat digunakan berulang.
   - `/api`: Serverless functions untuk menangani permintaan ke server.

6. **Cara Menjalankan Proyek:**

   - Install dependensi dengan perintah `npm install`.
   - Jalankan proyek dengan `npm run dev`.

7. **Rencana Pengembangan:**

   - Fase 1: Desain tata letak dan navigasi.
   - Fase 2: Implementasi fungsi registrasi dan login.
   - Fase 3: Pengembangan fitur jadwal kuliah dan pengumuman.

8. **Sumber Terpercaya:**
   - Next.js Documentation: [https://nextjs.org/docs/getting-started](https://nextjs.org/docs/getting-started)
   - MongoDB Documentation: [https://docs.mongodb.com/](https://docs.mongodb.com/)
   - JSON Web Tokens Introduction: [https://jwt.io/introduction/](https://jwt.io/introduction/)
